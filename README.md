# Soulspace Kids

Prototaip aplikasi web tempatan untuk menyokong pemerhatian emosi, intervensi mikro-spiritual, terapi seni geometri, pernafasan berpandu, dan laporan perkembangan murid pendidikan khas.

## Cara buka

Buka `index.html` menggunakan pelayar moden. Rekod sesi disimpan pada peranti melalui storan pelayar.

## Nota keselamatan

Modul kamera dalam prototaip ini ialah pratonton tempatan untuk pemerhatian guru. Ia tidak merakam video dan tidak mendiagnosis emosi murid secara automatik. Pengesahan manusia dan persetujuan penjaga/sekolah tetap diperlukan.

## AI Teachable Machine

Model lalai telah diset kepada `https://teachablemachine.withgoogle.com/models/4jBk5tQuGT/`.

Untuk tambah emosi lain, buka semula projek di Teachable Machine dan tambah kelas seperti `Marah`, `Sedih`, `Gelisah`, `Tenang`, dan `Fokus`. Masukkan contoh yang mencukupi untuk setiap kelas, tekan Train, kemudian export model. Jika Teachable Machine beri URL baharu, tampal URL itu dalam ruangan model di app.

## Audio

Modul bunyi menyokong rakaman zikir/Quran suara manusia melalui fail audio tempatan atau pautan YouTube, serta melodi lembut tanpa lirik. Pelayar biasanya memerlukan pengguna menekan butang dahulu sebelum audio boleh dimainkan.
